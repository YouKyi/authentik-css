# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-03-23

### Added
- Initial release
- Glassmorphism theme for Authentik 2026.x
- Login page: glass container, styled inputs, animated entrance
- Library page: glass cards, search bar, group headers with text-shadow
- SSO buttons with brand colors: Discord (purple), Plex (amber), Security Key (cyan)
- SSO button template for adding new providers
- Consent flow: green/red glass buttons
- 11 CSS theme variables for full customization
- 5 color presets: Blue, Red, Green, Purple, Neutral
- Admin hotfix: glass effects scoped to user-facing pages only
- Accessibility: prefers-reduced-motion, prefers-contrast, prefers-reduced-transparency
- Responsive: mobile-optimized with 48px touch targets
- Custom scrollbar (WebKit + Firefox)
- Notification/API drawer glass styling
- Dropdown menu glass styling
- Footer horizontal layout fix for ak-brand-links
